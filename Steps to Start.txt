

1) Open Windows
2) Search Anaconda prompt
3) Change directory to desired directory
4) type -> pycharm .         [This creates pycharm instance]



Create environment:
1) conda create -p venv python==3.8 -y   {creating environment with python version=3.8 and yes to all permissions}
2) conda activate venv/


To install requirements.txt file [packages install]
1) pip install -r requirements.txt


Git steps:
1) git add .  {To add files}
2) git reset  {Undo the git add}
3) git status {check status of added files}
4) git commit -m "Message"
5) git push -u origin main
